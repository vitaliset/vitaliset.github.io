# vitaliset.github.io

Personal blog for technical discussions on data science! The layout is a small
adaptation of the <a href="https://janczizikow.github.io/sleek/">Sleek</a> theme.

The site is live at <https://vitaliset.github.io/>.

## Development

Local setup, the repository layout, and how styles/assets are built are documented in
[`DEVELOPMENT.md`](DEVELOPMENT.md). The notebook → post pipeline lives in
[`scripts/README.md`](scripts/README.md).
